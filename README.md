# AI Attendance Management System

## Project Overview

AI Attendance Management System is a Java-based web application that uses OpenCV for face detection and automatic attendance marking.

## Features

- Student Management
- Login Authentication
- AI Face Detection
- Automatic Attendance
- Attendance Viewer
- OpenCV Integration
- MySQL Database
- Modern Dashboard UI

## Technologies Used

- Java
- JSP & Servlet
- MySQL
- OpenCV
- Apache Tomcat
- HTML/CSS
- GitHub

## Project Structure

```text
src/main/java/com/attendance
src/main/webapp
dataset
```

## Modules

### Login Module
Admin login system.

### Student Module
Add and view students.

### AI Attendance Module
Detect faces using webcam and mark attendance automatically.

### Attendance Viewer
View attendance records from database.
